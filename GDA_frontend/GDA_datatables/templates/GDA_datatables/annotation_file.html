Help

How to return to the homepage?

Use Home button for returning to the Homepage.

How to resize the table?

The table can be resized using a button next to the Home button, and there are three different sizes: small, medium, and large.

How to change the visibility of the columns?

Click on the Column visibility button and choose which column you want to hide or show.

How to extract data?

Click on the button Exact data and choose extraction format.

How to copy data from the table?

By pressing Copy button, there is an option to copy one or multiple rows. Also, if there are no rows selected current table page will be copied.

How to select rows?

Selection could be done with a mouse click on the row, if there is a reason for selecting all rows on the current page, then click the Select all button.

How to deselect rows?

Click on a row that you want to deselect, if you want to deselect all rows, then click on the button Deselect all.

How to change number of entries?

Click on the button for the number of entries and choose one of the options: 10, 25, 50, or 100.

How to search for data inside the table?

Enter a search term inside the search box.

How to search and order data by column?

For ordering, just click on the column which you want to sort. For searching, just enter a search term inside the search box for a column that you want to search.

How to use pagination?

You can choose Previous, Next, or a page close to the current page. Also, you can enter the page number and go straight to that page.
